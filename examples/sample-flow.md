# Sample Flow

1. Use `research-question-refiner` to turn the broad topic into a primary question and bounded sub-questions.
2. Use `search-string-designer` to create reproducible search strings and a source plan.
3. Use `paper-screening-checker` to apply inclusion/exclusion logic and log decisions.
4. Use `literature-theme-extractor` to cluster sources into themes and trends.
5. Use `methodology-comparer` and `evidence-strength-scorer` to compare approaches and confidence.
6. Use `gap-finder` to identify underexplored or weakly evidenced areas.
7. Use `critical-analysis-writer` and `related-work-synthesizer` to produce the narrative review sections.
8. Use `future-work-generator` to finish with grounded next-step recommendations.
